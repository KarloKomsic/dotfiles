## Install guide
This repo will assume you already have the necessary software installed, as to reduce the size of this guide.

You can clone this repo:

HTTPS:
```bash
git clone https://github.com/KarloKomsic/repository ~/Dotfiles
cd Dotfiles/
```

SSH:
```bash
git clone git@github.com:username/repository.git ~/Dotfiles
cd Dotfiles/
```
To install the dotfiles, head to the scripts/ directory:
```bash
cd scripts/
```
And then, run the install script:
```bash
./install.sh
```
